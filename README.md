# KeccakEx - NO OPTIMIZED

Small library for handling Keccak in Elixir.

## Usage

Add dependency in your mix.exs:

```elixir
def deps do
  [
    {:keccak_ex, "~> 0.1.0"}
  ]
end
```

Enjoy!
