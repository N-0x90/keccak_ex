# KeccakEx

Small library for handling Keccak 256/512 in pure Elixir.

## Usage

Add dependency in your mix.exs:

```elixir
def deps do
  [
    {:keccak_ex, "~> 0.4.0"}
  ]
end
```

Enjoy!
